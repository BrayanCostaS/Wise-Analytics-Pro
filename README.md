# WISE Analytics Pro 🚀
> Sistema de Auditoria e Inteligência de Dados para Gestão Educacional.

## ⚙️ Arquitetura do Sistema

O projeto foi desenvolvido utilizando uma abordagem de pipeline de dados completo:

- 🔎 Coleta de Dados: Extração automatizada (web scraping) de registros acadêmicos  
- 🧹 Tratamento (ETL): Limpeza e estruturação dos dados antes da persistência  
- 🗄️ Armazenamento: Banco de dados em nuvem  
- ⚡ Backend lógico: Regras de negócio para cálculo de progresso acadêmico  
- 💻 Frontend: Interface em React com visualização dinâmica  
- ☁️ Deploy: Aplicação hospedada no Netlify  

O **WISE Analytics Pro** é uma solução de auditoria de dados desenvolvida para resolver gargalos de produtividade no processamento de históricos escolares e créditos acadêmicos. O projeto transforma um fluxo de trabalho manual e exaustivo em uma operação automatizada, reativa e focada em insights.

---

## 📋 Contexto do Problema
O cenário anterior envolvia a conferência manual de mais de **3.000 registros**, exigindo múltiplos cliques e a navegação entre diversas abas para validar o progresso de cada aluno. 
* **Desafio:** Alta carga cognitiva e risco de erro humano na contagem de créditos.
* **Solução:** Automação da lógica de negócio para cruzamento instantâneo de dados.

## 🛠️ Stack Tecnológica
* **Front-end:** [React.js](https://reactjs.org/) (Single Page Application)
* **Visualização:** [Chart.js](https://www.chartjs.org/) para dashboards dinâmicos.
* **Estilização:** CSS3 / Styled Components (ou a biblioteca que você usou).
* **Infraestrutura:** Cloud Hosting com Pipeline de Deploy Contínuo.

## ✨ Funcionalidades Principais
- **Processamento Instantâneo:** Cálculo automático de progresso (ex: 2/4 unidades concluídas).
- **Segmentação Inteligente:** Filtros automáticos para identificar alunos em "Reta Final" e "Casos Críticos".
- **Dashboard de Créditos:** Diferenciação visual entre disciplinas obrigatórias e eletivas.
- **Interface Reativa:** Busca e filtragem de dados em tempo real sem recarregamento de página.

## 🏗️ Arquitetura e Engenharia
1.  **Higienização de Dados (ETL):** Levantamento e tratamento da base bruta para garantir integridade nos cálculos.
2.  **Lógica de Negócio:** Algoritmos desenvolvidos em JavaScript para validação de requisitos de graduação.
3.  **Hospedagem Segura:** Deploy em nuvem focado em disponibilidade e performance.

## 🛡️ Privacidade e LGPD
Este projeto foi desenvolvido estritamente para uso administrativo interno. 
> **Nota:** Em conformidade com a **Lei Geral de Proteção de Dados (LGPD)**, o repositório de código e o acesso à ferramenta são privados, garantindo o sigilo e a segurança de informações sensíveis e dados pessoais.
